# Acme::Umlautify

A port of Philip Pollard's perl Acme::Umlautify. It will add umlauts to everything, now including the capital Y!

## Installation

    git clone https://github.com/tallenaz/acme-umlautify.git
    cd acme-umlautify
    rake install

## Usage

    "Yes, Loincloth rocks!".umlautify!
    => "Ÿës, Löïnclöth röcks!"
